# AMA SESSION QUESTIONS AND ANSWERS:

## Adhikya Edammala: What is object-fit: cover  
`object-fit: cover` makes an image fill the container without stretching. Some parts may get cropped.

## Allanki VV Manikanta Sai: Difference between section and article  
`section` groups related content.
`article` contains independent content like a blog or news post.

## Arpit Yadav: Where can we specify script tag
`<script>` tag can be placed inside `<head>` or before `</body>`.

## Boorle Sowmya Sri Lakshmi: Difference between block, inline and inline-block 
`block` takes full width and starts on new line. 
`inline` stays in same line and cannot set width/height properly.
`inline-block` stays inline and width/height can be applied.

## Injamuri Arun Kumar: Difference between % and vw  
`%` is based on parent element size. 
`vw` is based on browser window width.

## Kamparapu Lakshman: Difference between flex-grow, flex-shrink, flex-basis  
`flex-grow` allows item to grow. 
`flex-shrink` allows item to shrink. 
`flex-basis` sets initial size of item.

## M Harivardhan Reddy: Difference between em and rem
`em` is relative to parent font size. 
`rem` is relative to root (`html`) font size.

## Md Musharaf: Difference between placeholder and value  
`placeholder` shows hint text inside input.
`value` stores actual input data.

## Naman Sharma: What is flex-wrap 
`flex-wrap` decides whether flex items move to next line or stay in one line.

## Nayunipatruni Harsha Vardhan: flex-direction and its values  
`flex-direction` sets direction of flex items. 
Values: `row`, `row-reverse`, `column`, `column-reverse`.

## Parlapalli Sulochana: Difference between space-between, space-evenly, space-around  
`space-between` gives space only between items. 
`space-around` gives space around items. 
`space-evenly` gives equal space everywhere.

## Rongala Vasu: Types of webpages 
- Static webpage
- Dynamic webpage

## Tumma Haritha: Purpose of float  
`float` is used to place elements left or right and wrap text around them.

## Vikas Mehta: What does target="_blank" do in anchor tag  
It opens the link in a new browser tab.

## Yash Nitin Raut: Is HTML a programming language  
No, HTML is a markup language used to structure webpages.
